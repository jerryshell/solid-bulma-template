# Solid Bulma Template

Preview: [https://solid-bulma-template.jerryshell.eu.org](https://solid-bulma-template.jerryshell.eu.org)

- TypeScript
- Vite
- SolidJS
- Bulma

## How to use

```bash
git clone https://github.com/jerryshell/solid-bulma-template --depth 1 my-project
```

```bash
cd my-project
```

```bash
rm -rf .git && git init
```

```bash
npm install
```

```bash
npm run dev
```
